Link para associar vídeos à imagens

Fazer um site com imagens thumb que ao clicar em qualquer uma delas
abra outra página com o vídeo relacionado. Os vídeos estão no YouTube.

  1. Crie a página principal do site
  2. Crie a página onde o vídeo vai estar
  3. Abra o vídeo no YouTube, clique em compartilhar depois
    em incorporar, clique em copiar e cole na tag da página do vídeo
  4. Na mesma página crie um link para voltar a página principal, para isso use a tag <a>
    copie o link URL da página principal e cole nesse link, dentro dessa mesma tag <a>
    coleque a imagem de uma seta para indicar retorno a página principal, use a tag <img> para fazer isso
  5. Copie o link URL da página do vídeo, vá até a página principal cole na tag <a> que indica esse vídeo
  6. Repita o processo para colocar 4 vídeos no site
